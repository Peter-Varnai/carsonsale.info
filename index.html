Exploring European Used Car Market Data

Welcome to carsonsale.info, the programmer portfolio project of Peter Varnai. This project involved web scraping data from well-known second hand car websites, to gain insights into car brand popularity across Europe. This data posed challenges due to inconsistencies and inaccuracies introduced by users of the websites. My focus was on extracting important data points, a sample of the raw data can be found here.

Data Enrichment and Mapping

As the project evolved, my primary focus shifted to creating a density map that visualizes the distribution of car brands across Europe. To achieve this, I used an open-source geolocation API to map city names and postal codes to latitude and longitude coordinates.

Backend Setup and API

I developed a FastAPI backend with an endpoint to receive from the front end the list of car brands the user would like to see, querying the database amassed by webscarping and returns the requested data precisely formatted to align with the expectations of the data visualizer library on the front end, ensuring seamless data presentation.

Key Skills and Tools

This project honed my skills in database management, web scraping, data preprocessing and crafting requests with tools like Insomnia. It showcased my ability to develop solutions for data cleaning, enrichment and visualization, demonstrating my abilities of data analysis, backend development and problem-solving.

Conclusion

Exploring and analyzing European used car market data revealed the challenges of working with real-world data and emphasized the significance of data preprocessing for meaningful insights. The resulting density map offers a unique perspective on car brand popularity in Europe.
